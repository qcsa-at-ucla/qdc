# Quantum Device Consortium Website

This is the official website for the Quantum Device Consortium (QDC), built with modern web technologies.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org) 16.0.1 (React 19.2.0)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4.x
- **Deployment**: Vercel (recommended for Next.js)

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## Project Structure

- `app/` - Next.js App Router directory containing pages and layouts
- `public/` - Static assets (images, etc.)
- `app/page.tsx` - Main landing page component
- `app/layout.tsx` - Root layout component
- `app/globals.css` - Global styles and Tailwind CSS configuration

## Features

- Responsive design with Tailwind CSS
- Interactive hover effects on buttons
- Optimized image loading with Next.js Image component
- TypeScript for type safety

## Build and Deploy

To create a production build:

```bash
npm run build
```

To start the production server:

```bash
npm start
```

The easiest way to deploy this Next.js app is to use the [Vercel Platform](https://vercel.com/new) from the creators of Next.js.
